#### NixOS
As it stereotypical of a computer nerd I use [Linux](https://en.wikipedia.org/wiki/Linux) as my main operating system, 
specifically [NixOS](https://nixos.org/), as I love the reproducibility it brings, alongside the hundreds of thousands of
packages. It means that I never forget some obsecure settings or configuration changes that I've made in some random file a
years ago.

However, the main reason I choose NixOS is for all the packages that it has, I love the fact that I never need to try
and figure out installing some software myself, as Nix takes care of it all for me. Even if I need to compile it myself,
Nix handles tedious processes, such as adding it to the path.
